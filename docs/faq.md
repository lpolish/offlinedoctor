# Frequently Asked Questions

## üìã General Questions

### What is Offline Doctor?
Offline Doctor is an AI-powered medical assistant that runs completely offline on your computer. It provides general medical information and guidance while ensuring your privacy through local-only processing.

### Who is it for?
* General users seeking medical information
* Healthcare professionals for patient education
* Clinics and medical practices
* Anyone prioritizing privacy in healthcare

### Is it a substitute for medical professionals?
No. Offline Doctor is an informational tool only. Always consult qualified healthcare professionals for medical diagnosis, treatment, and care decisions.

## üîí Privacy & Security

### How is my data protected?
* All processing happens locally
* No internet connection required
* Optional encryption for stored data
* Regular automatic data cleanup

### Can I delete my data?
Yes, you have full control:
* Clear individual sessions
* Delete all history
* Configure auto-deletion
* Export data before removal

### Is my data shared with anyone?
No. Your medical information never leaves your device. The application runs completely offline after initial setup.

The AI provides general medical information based on its training. Accuracy varies, and all information should be verified with healthcare professionals.

## üõ†Ô∏è Troubleshooting

### Common Installation Issues

**Windows Installation Failed:**
1. Run as administrator
2. Check Windows Defender
3. Verify system requirements
4. Try portable version

**macOS "App Cannot Be Opened":**
1. Open System Preferences
2. Go to Security & Privacy
3. Click "Open Anyway"
4. Follow security prompts

### Application Problems

**App Won't Start:**
* Check Node.js version
* Verify Python setup
* Confirm Ollama status
* Check system resources

**AI Model Issues:**
* Verify model installation
* Check Ollama service
* Restart application
* Reinstall if needed
